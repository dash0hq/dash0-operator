# Dash0 Operator Documentation Structure

This directory contains the split documentation for the Dash0 Operator Helm Chart.

## Documentation Files

1. **INSTALLATION.md** - Installation procedures and prerequisites
2. **CONFIGURATION.md** - Configuration options for operator and monitoring resources
3. **AUTO-INSTRUMENTATION.md** - Automatic workload instrumentation details
4. **METRICS-AND-SCRAPING.md** - Metrics collection and Prometheus scraping
5. **MANAGING-DASH0-RESOURCES.md** - Infrastructure-as-code for Dash0 resources
6. **ADVANCED-CONFIGURATION.md** - Advanced configuration options
7. **PLATFORM-SPECIFIC.md** - Platform-specific notes and compatibility
8. **UPGRADING.md** - Upgrade and uninstallation procedures
9. **TROUBLESHOOTING.md** - Troubleshooting and debugging

## Purpose

The original README.md exceeded Google's 2MB indexing limit (192KB). This restructuring:
- Keeps each page well under the limit
- Improves navigation and discoverability
- Organizes content by topic
- Maintains SEO indexing
